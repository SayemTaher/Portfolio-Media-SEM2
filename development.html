Challenge One: Landing Page of Aircompany.com

Task: The first task assigned to us was to design the landing page for the Aircomapny. It's a simple design that allows students to experiment with different ways of creating a navigation bar while the end result should remain consistent. Students will also learn how to add icons beside the tab title and how to use a gradient background on a button in this challenge. I created the nav bar with CSS Flexbox, which allowed me to give the navigation the proper shape. Though I couldn't find the right video to add to the homepage's main section, I used  similar gradient, so it may appear similar.

Learning Outcome: While working on this challenge, I discovered new ways to design navigation bars. Despite the fact that I had a lot of experience with this type of layout, it was a little easier for me. When I first saw the structure, I knew exactly how I should approach it. This assignment assisted me in reviewing my previous CSS knowledge.

Challenge Two: Generate Image On Click

Task: We had to generate an image while clicking on any part of a webpage during this challenge. It was the first challenge where we were advised to use JavaScript to achieve the desired result using an event handler. The first step was to generate an image by clicking on a button. Later, my teacher advised us to take it a step further and generate an image by clicking. Essentially, it was intended to test our understanding of how the click event works on a webpage.

Learning Outcome: It was a unique challenge. Working on this challenge helped me better understand how a webpage's click event works. It was our first assignment in which we used JavaScript. It's intriguing that using JS, you can do anything and manipulate the structure of a webpage. Rather than focusing solely on achieving the goal, my goal for this challenge was to understand the behavior of JavaScript. Finally, this assignment taught me how to work with image loading and click events.

Learning Outcome: It was my first challenge in which I had to work with a single HTML file. But I had a great time and learned a lot about working with websites. Also, each time I clicked on a different navbar, I had to change the CSS file. During that time, I learned different ways to work with JS and how to achieve the desired result. Overall, I was pleased that I could do it; it was both challenging and interesting.

Challenge Four: Useful ways of using CSS Animation

Description: We were assigned the task of experimenting with different ways of working with CSS animation on our portfolio website. We were able to use fantastic animation on our website to give it a more cool feeling this way. Any user's entire website experience can be improved by animation. On my portfolio, I used several animations such as typewriter animation, wave animation, and image 3D carousel. The one I loved most was the 3D image carousel, which I used on my landing page.

Animation Explained: This is one of the more intricate animations I've worked on up to this point. Inspired by the Online Tutorials channel on YouTube, I wanted to incorporate it into my portfolio. To me, it appears quite nice to slide over different graphics to make the page more engaging. For each label to trigger when checked, I had to use multiple label & id's . When a label is checked based on its position, I used transform:translate3D() to adjust the position of the card. This is how I accomplished three-dimensional transformation as you can see in the code excerpt, different values are returned each time a label is checked.

Learning Outcome: I wanted to explore different methods of CSS animation from the beginning of this semester. To comprehend the behavior. This only showed to me that CSS is far more powerful than it appears and that there is a huge variety of possibilities that can be achieved using simply CSS. This assignment assisted me in understanding and going over my prior implementations to determine the WHY.

Chrome Extension

Description: The Chrome extension is a straightforward application that allows you to capture and store any Chrome URL, enabling you to save URLs of significant importance or interest. This functionality can be achieved by utilizing the "Save Tab" button, which automatically saves the current URL. Additionally, you have the option to manually enter a URL name and save it to the browser's local storage. To delete the stored data, simply double-click the "Delete All" button. Prior to utilizing the extension, it is necessary to deploy it on chrome://extensions while enabling developer mode.

Code Snippet

Learning Outcomes: I have learned a great deal through this exercise, particularly in utilizing local storage as a simplified database for data storage and retrieval. Additionally, I gained knowledge on deploying extensions on Google Chrome. The JavaScript course provided by Scrimba was instrumental in enhancing my understanding of fundamental concepts such as arrays, functions, conditionals, DOM manipulation, object manipulation, and more.
